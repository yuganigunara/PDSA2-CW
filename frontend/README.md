# Knight's Tour React Frontend

This folder contains a React + Vite frontend for the Knight's Tour project.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
cd frontend
npm run build
```

## What it includes

- Responsive React UI
- Manual knight-tour play mode
- Solver animation with Warnsdorff and backtracking options
- Local winner snapshots stored in browser storage
- Mobile-friendly layout and board visualization
